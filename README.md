### Import DB:

1. ```$ mysql -u root -p```
2. ```mysql> CREATE DATABASE SISTEMA_LOJA;```
3. ```mysql> quit```
4. ```$ mysql -u root -p SISTEMA_LOJA < SISTEMA_LOJA.sql```

### Execute:
  
```
$ cd GerenciadorCompras
$ python3 Inicio.py
```
